# 🧊 NM Refrigeração - Sistema Integrado

## Visão Geral

Sistema web completo integrado para gerenciamento de negócio de refrigeração e ar condicionado. Combina funcionalidades de admin e cliente em uma única plataforma, com suporte para:

- ✅ Gestão de Clientes
- ✅ Catálogo de Produtos e Serviços
- ✅ Orçamentos e Pedidos
- ✅ Agendamentos com Calendário
- ✅ Vendas e Cobranças
- ✅ Garantias e Manutenção Preventiva (PMP)
- ✅ Relatórios Técnicos com IA
- ✅ Dashboard Financeiro
- ✅ Geração de PDFs
- ✅ Integração WhatsApp
- ✅ Assistente IA para Diagnósticos

## 🚀 Instalação Rápida

### 1. Pré-requisitos

- PHP 7.4 ou superior
- MySQL 5.7 ou superior
- Extensões PHP: MySQLi, GD, cURL, JSON
- Navegador moderno (Chrome, Firefox, Safari, Edge)
